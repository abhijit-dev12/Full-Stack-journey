# HTML Notes - Day 4

## Table

A table is used to display data in rows and columns.

```html
<table></table>
```

---

## Table Row

```html
<tr></tr>
```

Creates a row.

---

## Table Header

```html
<th></th>
```

Displays bold centered headings.

---

## Table Data

```html
<td></td>
```

Stores actual data.

---

## Table Head

```html
<thead></thead>
```

Contains column headings.

---

## Table Body

```html
<tbody></tbody>
```

Contains main data.

---

## Table Footer

```html
<tfoot></tfoot>
```

Contains totals or summary.

---

## Border

```html
border="1"
```

Adds a border.

---

## Cell Padding

```html
cellpadding="10"
```

Space inside each cell.

---

## Cell Spacing

```html
cellspacing="0"
```

Space between cells.

---

## Colspan

Merge columns.

```html
<td colspan="3">
```

---

## Rowspan

Merge rows.

```html
<td rowspan="2">
```
