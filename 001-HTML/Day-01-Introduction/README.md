# HTML Notes - Day 1

## What is HTML?

HTML stands for HyperText Markup Language.

It is used to create the structure of web pages.

HTML is not a programming language.
It is a markup language.

## What is HyperText?

HyperText means text that contains links to other pages.

## What is Markup?

Markup tells the browser how content should be displayed.

Example:

<h1>Heading</h1>

## HTML File Extension

.html

## HTML was created by

Tim Berners-Lee

Year: 1991

## HTML Document Structure

DOCTYPE

html

head

body

These four are the foundation of every HTML page.
