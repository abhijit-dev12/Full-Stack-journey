# HTML Notes - Day 3

## Unordered List

Displays items with bullets.

```html
<ul>
    <li>Apple</li>
    <li>Mango</li>
</ul>
```

---

## Ordered List

Displays numbered items.

```html
<ol>
    <li>Wake Up</li>
    <li>Study</li>
</ol>
```

---

## Description List

Used for definitions.

```html
<dl>
    <dt>HTML</dt>
    <dd>Markup Language</dd>
</dl>
```

---
