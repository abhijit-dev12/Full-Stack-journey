# HTML Notes - Day 2

## Headings

HTML provides six levels of headings.

```html
<h1>Main Heading</h1>
<h2>Heading</h2>
<h3>Heading</h3>
<h4>Heading</h4>
<h5>Heading</h5>
<h6>Heading</h6>
```

Use only one `<h1>` on a page because it represents the main title.

---

## Paragraph

```html
<p>This is a paragraph.</p>
```

Paragraphs automatically have spacing before and after them.

---

## Line Break

```html
<br>
```

Moves the next content to a new line.

---

## Horizontal Rule

```html
<hr>
```

Creates a horizontal line.

---

## Bold vs Strong

```html
<b>Bold</b>

<strong>Important</strong>
```

- `<b>` makes text bold.
- `<strong>` makes text bold and indicates importance (better for accessibility and SEO).

---

## Italic vs Emphasis

```html
<i>Italic</i>

<em>Emphasized</em>
```

- `<i>` changes appearance.
- `<em>` adds emphasis with semantic meaning.

---

## Subscript

```html
H<sub>2</sub>O
```

Displays text below the normal line.

---

## Superscript

```html
10<sup>2</sup>
```

Displays text above the normal line.
