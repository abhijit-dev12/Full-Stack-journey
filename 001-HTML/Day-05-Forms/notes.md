# HTML Notes - Day 5

## Form

Used to collect user input.

```html
<form></form>
```

---

## Label

```html
<label>Name</label>
```

Provides a title for an input.

---

## Input

```html
<input type="text">
```

Creates an input field.

---

## Input Types

- text
- email
- password
- number
- date
- radio
- checkbox

---

## Select

```html
<select>
    <option>India</option>
</select>
```

Creates a dropdown list.

---

## Textarea

```html
<textarea></textarea>
```

Used for multi-line text.

---

## Button

```html
<button>Submit</button>
```

---

## Required

```html
required
```

Prevents form submission if empty.

---

## Placeholder

```html
placeholder="Enter your name"
```

Shows helper text inside the input.
